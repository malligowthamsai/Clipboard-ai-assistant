# 🚀 ClipCode AI (Java)

## 🧠 Overview
ClipCode AI is a Java-based productivity tool that reads coding questions directly from your clipboard and generates clean, runnable code using AI.

Just copy a question → press Enter → get code instantly.

---

## ✨ Features
- 📋 Reads clipboard input
- 🤖 Generates code using AI (Google Gemini / OpenAI)
- ⚡ Fast and lightweight (CLI-based)
- 🎯 Returns only clean code (no explanations)
- 🔄 Continuous usage loop

---

## 🛠 Tech Stack
- Java
- HttpClient (Java 11+)
- Clipboard API (AWT)
- AI APIs (Google Gemini / OpenAI)

---

## ▶️ How It Works
1. Copy a coding question
2. Run the program
3. Press ENTER
4. Get AI-generated code instantly
